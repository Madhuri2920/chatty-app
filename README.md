# Chatty App

A full-stack social media app.

- **Frontend:** React (`socialmedia/`)
- **Backend:** Node.js + MongoDB (`chatty-backend/`)

## Run the project

### Backend

cd chatty-backend
npm install
node server.js

### Frontend
cd socialmedia
npm install
npm run dev
